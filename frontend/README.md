# Perion Task Frontend

This is the Vue 3 frontend for the RBAC-test project.

## Features

- Login form and auth state
- Dashboard with role and user management
- Permission toggling and data tables
- API client hooks under `src/api`

## Notes

- Backend API is expected at `http://localhost:3000` by default
- Use the provided seed data from the backend to login and test roles/permissions
